# Note Manager

A simple command-line Note Manager built with Python. This application allows users to create, view, open, edit, rename, search, and delete text notes using Python file handling.

## Features

* Create Notes
* View All Notes
* Open Notes
* Edit Notes
* Rename Notes
* Delete Notes
* Search Notes
* View Statistics
* Automatic Notes Folder Creation
* Input Validation

## Technologies Used

* Python 3
* File Handling
* OS Module
